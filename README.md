# BraidTrack 360: Operator Industrial Hub

BraidTrack 360 is a high-end SaaS industrial monitoring platform designed for smart factory management. It provides a comprehensive suite of tools for real-time analytics, machine health tracking, and operator performance metrics, all wrapped in a futuristic, high-performance interface.

## 🚀 Features

### 🔐 Futuristic Login Experience
- **Industrial Visuals**: Highly animated 3D wireframe machine simulation.
- **Digital HUD**: Real-time display of system status, RPM, temperature, and voltage.
- **Secure Access**: Role-based entry for Administrators and Operators.

### 🛠️ Admin Dashboard
- **Machine Inventory**: Add, configure, and decommission machines with ease.
- **Operator Management**: Comprehensive onboarding for operators with shift and machine assignments.
- **Revenue Analytics**: Track daily revenue, weekly forecasts, and efficiency rates with smooth, animated counters.
- **System Reports**: Generate and download production summaries, downtime analysis, and efficiency reports.
- **Factory Parameters**: Global control over production targets and alert thresholds.

### 👷 Operator Interface
- **Desktop-View Dashboard**: Optimized for factory floor workstations.
- **Issues Tracking**: Real-time table for reporting and resolving machine faults.
- **Shift Summary**: Instant access to total production, downtime, idle time, and efficiency metrics.
- **Notifications Hub**: Critical alerts for target variances and machine idle states.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion (motion)](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd react-example
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`.

## 📜 Scripts

- `npm run dev`: Starts the development server on port 3000.
- `npm run build`: Builds the application for production.
- `npm run lint`: Runs TypeScript type checking.
- `npm run preview`: Previews the production build locally.

## 🎨 Design Philosophy

BraidTrack 360 follows a **Technical Dashboard** aesthetic:
- **Dark Theme**: Deep navy backgrounds (`#05070a`) for reduced eye strain in industrial environments.
- **Neon Accents**: High-visibility neon green (`#00ff9d`) and orange (`#ff9d00`) for critical data points.
- **Glassmorphism**: Layered transparency and backdrop blurs for a premium, modern feel.
- **Precision Typography**: JetBrains Mono for technical data and Inter for UI elements.
